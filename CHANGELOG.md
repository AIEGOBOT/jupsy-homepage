## Build 0.3.0 - 2026-04-17

- Added: `WORKS` 맨 마지막에 강의/행사 이미지 3건을 추가하고, 각 항목을 내부 상세 페이지로 연결했습니다.
- Changed: 새 강의/행사 3건의 상세 갤러리를 실제 폴더 기준 전체 컷으로 반영했으며, 경북 교육자 대상 AI 역량강화 강의 5장, 영산대 AI 웹툰 강의 4장, RISE 사업특수효과(VFX) 해커톤 및 AI 상영 6장으로 구성했습니다.
- Changed: 홈, 소개, 작업 상세 페이지의 SEO 메타데이터와 구조화 데이터(JSON-LD)를 보강하고, 기본 공유 이미지를 동적 OG 이미지 라우트 기준으로 정리했습니다.
- Changed: `npm run dev`를 `webpack` 모드로 전환하고, 모바일 `WORKS` 그리드는 최적화를 위해 카드 모션 및 영상 자동재생을 끄도록 조정했습니다.
- Fixed: 르플레르 구강 스프레이 상세페이지 갤러리는 파일명 숫자 기준 정렬로 노출 순서를 안정화했습니다.
- Notes: Windows 한글 경로 환경에서는 개발 서버를 `npm run dev`로 실행하면 `webpack` 기준으로 동작합니다.

## Build 0.2.0 - 2026-04-13

- Added: `WORKS`에 고려대학교 봉사단 홍보 영상, 리베리 의원 강남점 스크린 영상, 정성곳간 프리미엄 총각김치 상세페이지, 르플레르 구강 스프레이 상세페이지를 추가했습니다.
- Changed: 신규 영상 2건을 홈 포트폴리오용 `webm` 미리보기로 변환하고, 신규 이미지 2건의 썸네일 및 상세페이지 에셋을 `webp`로 정리했습니다.
- Fixed: 이미지 상세페이지 원본에서 GIF를 제외하고 정적 이미지 중심으로 갤러리를 구성했으며, 상세페이지 갤러리 이미지는 원본보다 확대되지 않도록 표시 방식을 조정했습니다.
- Notes: 현재 기준 신규 상세페이지 갤러리는 프리미엄 총각김치 14장, 르플레르 구강 스프레이 30장으로 반영했습니다.
