# 홈페이지 기획 정리

## 1. 목적
- 주목적: 회사 소개
- 부목적: AI 이미지 제작 / AI 영상 제작 의뢰 수주
- 성격: 소개형 홈페이지가 중심이고, 문의 전환은 명확하게 보이게 구성

## 2. 전체 메뉴
- ABOUT
- WORKS
- CONTACT

## 3. 메인 페이지 핵심 방향
- 첫 화면에서 사용자가 바로 이해해야 하는 것:
  - 이 회사가 무엇을 하는지
  - 어떤 의뢰를 받을 수 있는지
  - 어디로 문의하면 되는지
- 메인에서 가장 중요한 행동 유도:
  - AI 이미지 제작 의뢰
  - AI 영상 제작 의뢰
- 위 2가지는 큰 카드 또는 큰 버튼 형태로 배치
- 클릭 시 각 의뢰 페이지 또는 문의 작성 영역으로 이동

## 4. 메인 페이지 구성 초안
### Hero
- 회사 한 줄 소개
- 서브 카피: 빠르고 효율적인 AI 기반 제작
- 대표 비주얼
- 주요 CTA 2개:
  - AI 이미지 제작 의뢰
  - AI 영상 제작 의뢰

### Why Section
- 왜 접시인가? / 왜 우리인가?
- 핵심 메시지 초안:
  - 제작비와 시간을 줄일 수 있다
  - 제작 효율을 높일 수 있다
  - 기존 제작 과정의 고질적인 문제를 해결할 수 있다

### Works Preview
- 납품 작업물 소개
- 자체 뷰티컷 / 샘플 결과물 강조
- 대표 작업물 썸네일 노출
- 인스타그램 링크 연결 가능

### Contact CTA
- 의뢰 문의 유도 문구
- 빠른 상담 또는 프로젝트 문의 버튼

## 5. ABOUT 페이지
- 회사 소개
- 어떤 작업을 하는 회사인지 설명
- AI 기반 이미지 / 영상 제작 역량 소개
- 강점 정리:
  - 빠른 제작
  - 비용 절감
  - 높은 활용도
  - 실무형 결과물

## 6. WORKS 페이지
- 실제 납품 작업물
- 자체 제작한 뷰티컷 / 샘플 작업물
- 작업 카테고리 구분 가능:
  - 이미지
  - 영상
- 인스타그램 링크 연결 검토

## 7. CONTACT 페이지
- 문의 폼 필수 항목 초안:
  - 연락처
  - 이메일
  - 희망 단가
  - 의뢰 내용
  - 수량 또는 컷 수
  - 영상일 경우 길이
  - 희망 납기일
- 문의 유형 선택:
  - AI 이미지 제작 의뢰
  - AI 영상 제작 의뢰

## 8. 디자인 메모
- 디자인 방향은 Google Stitch를 활용해 대략적인 시안 탐색
- 현재 단계에서는 구조와 메시지를 먼저 확정
- 메인에서 두 서비스 카드가 가장 먼저 보이도록 설계

## 9. 추후 검토 항목
- 영어 버전 제공 여부
- PayPal 결제 도입 여부
- 의뢰 버튼 클릭 후 이동 방식
  - 별도 상세 페이지
  - CONTACT 폼으로 스크롤 이동
  - 팝업 / 모달
